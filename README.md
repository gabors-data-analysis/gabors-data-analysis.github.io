## Website

[gabors-data-analysis.com](https://gabors-data-analysis.com/) — chapters, case studies, datasets, code, and resources for instructors and students.

## How this repo works

- Jekyll site built and served by GitHub Pages (branch: `master`).
- Content lives in `_pages/`, with collections in `_chapters/`, `_casestudies/`, `_datasets/`, and blog posts in `_posts/`.
- Navigation is defined in `_data/navigation.yml`.
- `ai-course/` and `book/` are rendered Quarto artifacts deployed from separate repositories — do not edit them here.
